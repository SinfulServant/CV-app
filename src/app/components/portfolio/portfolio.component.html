ChEng Island

Site learning English and Chinese, which was developed by a small team on freelance. Written with Angular and Nest. I independently made the entire frontend part of the application.

The application has about 10 pages, two languages, forms and validation, some auxiliary libraries. There is an admin panel where you can add and change the content of the page in real time.

In this project, I gained a lot of experience in using various Angular entities, tried to optimize code and architecture. I also interacted with the server a lot, used Swagger.
